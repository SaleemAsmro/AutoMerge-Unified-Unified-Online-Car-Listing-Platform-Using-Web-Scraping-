# AutoMerge-Unified-Unified-Online-Car-Listing-Platform-Using-Web-Scraping-
A central website that has scraped listings from other online car listing websites aggregated into one place for convenience and ease of use. This is a univeristy project made public for other people who may benefit from this.



Hardware needed: 
Computer + Internet connection



Software needed:
1- Python IDE (e.g Pycharm) + Selenium installation via terminal

2- ChromeDriver 

3- Any IDE that has live server for hosting a website. (e.g VS Code)





Steps to use:

(OPTIONAL) [since the json file already exists its optional]

1- From the following folders: CarsIqScraper, IqCarsScraper, KurdShoppingScraper, locate the _main.py_ file

2- In the terminal of your Python IDE, type _pip install selenium_ and press enter.

3- Assuming you have chromedriver installed, run the _main.py_ file.

4- Let the script do its thing

(How to run the website)

5- **IMPORTANT** Go to _Web/behavior/utils.js_ and change the paths of the JSON files found in _web/data/CarsIq.json_, _web/data/IQcars.json_, and _web/data/KurdShopping.json_ to where the files are actually located in your computer. 

5- Open the folder _Web_ in your IDE.

6- Launch _index.html_

7- Enjoy
